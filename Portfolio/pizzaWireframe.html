Pizza Delivery Wireframe

This wireframe presents the structural blueprint of a comprehensive pizza delivery website, embodying the essential features pivotal to its functionality and user experience. The site encompasses crucial pages including "Deals," "Menu," "About Us," "Contact Us," "Cart," and "Account," each meticulously designed to serve distinct purposes and deliver a seamless browsing experience for visitors.

Website Wireframe

This student learning portal is made following closely with this wireframe. With the easy navigation at the top that allow user to navigate around different places throughout the website. The website uses cards to showcase the courses available for enrolment. Cards are placed in a carousel that showcases more courses.

Color Palette

The colour scheme used is an analogous colour scheme with the main colour being red, with the a orangey red colour (#C23704) and purplish colour (#CC0479). The analogous colour scheme was chosen to give the website a harmonious look. Since analogous colour scheme uses colours adjacent to each other, they complement each other well giving the website a harmonious visual appearance.
